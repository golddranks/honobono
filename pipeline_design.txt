(ch1)

- char_extract: extracts a character list from ch1
	- inputs: ch1 text
	- outputs: character list
- char_extract_verify: asks if the list is missing any charcters
	- inputs: character list, ch1 text
	- outputs: bool -> crash early if false
- char_update_reg_a (non-LLM)
	- inputs: character list
	- outputs: updated character registry
- summarize: summarizes the ch1
	- inputs: ch1 text, character registry
	- outputs: summary text

(ch2)

- char_extract: extracts a character list from ch2
	- inputs: ch1 text, character registry, ch2 text
	- outputs: character list
- char_extract_verify: asks if the list is missing any charcters
	- inputs: ch1 text, character list, ch2 text
	- outputs: bool -> crash early if false
- loop for characters (tight loop to reuse the query cache)
	- char_merge_eval: evaluates if character is likely to be same with each character in the registry
		- inputs: ch1 text, character registry, ch2 text, target character
		- outputs: eval for each character in the registry
- loop for characters
	- char_merge_prune_candidates (non-LLM): picks only the likely match candidates from the character registry
		- inputs: character registry, character eval
		- outputs: pruned character registry
	- char_merge_judge: judges if the character matches some of the candidates, or is a new character
		- inputs: ch1 text, target character, pruned character registry, ch2 text
		- outputs: single-character judgement
- loop for new characters
	- char_merge_prune: removes any new characters that are unlikely to be actual characters
		- inputs: ch1 text, target character, character registry, ch2 text
		- outputs: pruning decision
- char_update_reg_b (non-LLM)
	- inputs: character registry, character list, character judgements, pruning decisions
	- outputs: updated character registry
- loop for updated characters:
	- consolidate_char: consolidates the merged character info (descriptions, canonical name, other names)
		- inputs: ch1 text, ch2 text, character registry, target_character
		- outputs: consolidated character info
	- update_char_history: updates the character identity history with the consolidated character info
		- inputs: ch1 text, ch2 text, character registry, target_character
		- outputs: updated character registry
- char_update_reg_c (non-LLM)
	- inputs: character registry, character list, character judgements, pruning decisions
	- outputs: updated character registry
- summarize: summarizes the ch2
	- inputs: ch1 text, character registry, ch2 text
	- outputs: summary text
